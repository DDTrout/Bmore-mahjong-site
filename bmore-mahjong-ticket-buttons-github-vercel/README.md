# Bmore Mahjong Website

This is a Vite + React one-page website for Bmore Mahjong.

## Upload to GitHub

Upload the contents of this folder directly into a new GitHub repository. The repo should show `package.json`, `index.html`, `vite.config.js`, `src`, and `public` at the top level.

## Deploy on Vercel

Import the GitHub repo into Vercel. Use these settings:

- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

Test the `.vercel.app` URL before connecting the custom domain.
